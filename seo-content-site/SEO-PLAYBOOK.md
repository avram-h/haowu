## 一、关键词策略

### 1.1 关键词金字塔

```
                    ┌─────────────┐
                    │  品牌词      │  ← 流量大、竞争激烈、转化高
                    │  "索尼耳机"  │
                    ├─────────────┤
                    │  品类大词    │  ← 搜索量最大、最难排
                    │  "蓝牙耳机"  │
                    ├─────────────┤
                    │  长尾词      │  ← 竞争小、转化高、适合新站
                    │  "500元以内   │
                    │   降噪耳机"   │
                    ├─────────────┤
                    │  问题词      │  ← 零竞争、精准流量
                    │  "蓝牙耳机    │
                    │   一只不响    │
                    │   怎么办"     │
                    └─────────────┘
```

**新站策略**：80% 长尾词 + 20% 大词。先拿长尾词拿到排名和流量，再攻大词。

### 1.2 关键词挖掘方法

```bash
# 方法1：百度/Google 搜索下拉框
# 搜索 "蓝牙耳机" → 看联想词 → 全是长尾金矿

# 方法2：5118.com / 爱站网 长尾词挖掘
# 免费版每天可查几十个

# 方法3：竞争对手分析
# 找同行网站 → 看他们在排什么词 → 跟着做
```

### 1.3 好关键词的判断标准

| 指标 | 好 | 坏 |
|------|----|----|
| 搜索量 | 100-3000/月 | <50 或 >10万 |
| 竞争度 | 低-中（前10无大站） | 前10全是京东天猫 |
| 商业意图 | 有（"推荐""多少钱""哪个好"） | 纯信息类（"是什么""历史"） |
| 能否产出内容 | 能写 800 字以上 | 一句话就能说完 |

## 二、内容策略

### 2.1 文章类型配比

| 类型 | 占比 | 目的 | 示例 |
|------|------|------|------|
| 🏆 推荐榜单 | 30% | 拿长尾词流量 | "2024年蓝牙耳机推荐TOP10" |
| 📝 单品评测 | 25% | 建立权威性 | "AirPods Pro 2 深度评测" |
| ⚖️ 对比评测 | 20% | 截获对比搜索 | "AirPods vs Sony 全面对比" |
| 🛠️ 使用教程 | 15% | 长尾加粉 | "蓝牙耳机连接不上怎么办" |
| 💡 选购指南 | 10% | 入口内容 | "第一次买降噪耳机怎么选" |

### 2.2 文章 SEO 检查清单

每篇文章发布前检查：

- [ ] 标题包含核心关键词（前15个字内）
- [ ] Meta Description 包含关键词 + CTA（150字内）
- [ ] URL 包含关键词拼音/英文
- [ ] H1 = 标题，H2 至少 3 个
- [ ] 首段 100 字内出现核心关键词
- [ ] 文中关键词密度 1-2%（自然分布）
- [ ] 至少 1 张相关图片（带 alt 标签）
- [ ] 至少 2 个内链（链接到本站其他文章）
- [ ] 至少 1 个外链（链接到权威来源）
- [ ] 字数 > 800（越长越好，2000+ 更佳）
- [ ] 包含 FAQ 区块（命中 People Also Ask）
- [ ] 结构化数据正确（Product/Article/FAQ）

## 四、外链策略

外链是 SEO 的"选票"。每一条高质量外链都在告诉搜索引擎：这个网站值得信赖。

### 4.1 免费外链来源

| 渠道 | 难度 | 效果 |
|------|------|------|
| 知乎回答带链接 | ⭐ | ⭐⭐ |
| 小红书笔记 | ⭐⭐ | ⭐ |
| CSDN/掘金/博客园 | ⭐ | ⭐⭐ |
| GitHub README 外链 | ⭐ | ⭐ |
| 友情链接交换 | ⭐⭐ | ⭐⭐ |
| 行业论坛签名 | ⭐ | ⭐ |
| 维基百科外部链接 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| HackerNews/ProductHunt | ⭐⭐⭐ | ⭐⭐⭐ |

### 4.2 知乎引流 SOP

1. 搜索你文章领域的问题（如"蓝牙耳机怎么选"）
2. 写 500 字高质量回答（认真写，别水）
3. 文末自然插入："更多推荐可以参考这篇文章 [链接]"
4. 每天回答 2-3 个问题
5. 坚持一个月，效果明显

## 八、常见问题

**Q：多久能看到流量？**

新域名通常 3-6 个月开始有明显自然流量。前 3 个月可能每天只有个位数访客，这是正常的"沙盒期"。

**Q：需要写多少篇文章？**

50 篇是及格线，100 篇是分水岭，200 篇以上开始有规模效应。

**Q：中文 SEO 和英文 SEO 哪个好做？**

中文搜索百度占主导，规则不同（更看重域名年龄和备案）。Google 中文搜索竞争相对小，建议初期双管齐下。

**Q：要不要做视频内容？**

文字 SEO 做起来后再考虑。YouTube 是第二大搜索引擎，但视频制作成本高。先用文字跑通模型。

**Q：被百度/Google 降权怎么办？**

1. 检查是否有垃圾外链 → 拒绝
2. 检查是否内容重复 → 加 canonical
3. 检查是否有过度优化 → 降低关键词密度
4. 坚持原创、稳定更新 → 通常 2-4 周恢复
